Python之路

2020-12-28 2 min read # Python笔记

SuoNi学习python

一.概述

1.对集成环境的安装，在涛学长的推荐下使用了pycharm，并且进行社区版安装与汉化。
2.学习基础的语法，这里根据B站的《马士兵python入门视频》进行学习，个人感觉由于已经有了对C++以及其IDE一些知识的了解，学起来并不是那么费劲。
3.这份博客更多的是为了自己以后学习了很多语言，不混淆基本语法，然后作为漫漫长路陪伴自己走下去的一个见证吧，在边学习边这样记录中，目前是有一种想法把python作为一种主攻方向并坚持下去的。

（2020/12/28）

二.学习概况

1.对编译器的熟悉与print（）函数的简单使用：

print("good")
cool=open('D:/这是一个word文档.doc','a+')
print('hellow',file=cool)
cool.close()

这里我调用了函数来在D盘目录下开创了一个.doc类型的文件并且把“hellow”写入其中，我确实成功的找到了这个文件，并且对了，也在操作台显示出了“good”。
